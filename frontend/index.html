使用文档

快速开始

羽刃创意中心是一款面向电商视觉团队的 AI 生图工作台，支持四种生图模式，覆盖从主图到详情页、从开品设计到视频脚本的全链路需求。

基本流程

  1. 在顶部导航选择模式（自定义 / 电商 / 开品 / 视频）
  2. 在右侧面板配置参数（品类、卖点、参考图等）
  3. 在底部输入框补充附加要求，点击发送
  4. 等待生成完成，点击 💾 保存到本地画廊
提示 — 生成的图片会先进入临时归档（2 小时后自动清理），点 💾 才会永久保存到你在设置里指定的目录。

选择模型

每种模式都可以在发送前切换 AI 模型。不同模型在风格、速度、成本上有差异，建议先用默认模型出图，满意后再批量生成。

四种模式

自定义模式

最灵活的模式，适合有明确 prompt 想法的用户。可以直接输入文字描述，也可以上传参考图让 AI 在此基础上重绘或扩展。

  • 产品类目 — 点击"选择类目"按钮，从树状菜单中选择，支持搜索。选中后系统会自动补充该品类的风格约束。
  • 上传图片 — 拖拽或点击上传区域，支持多张。上传后图片作为参考底图，AI 会在保持产品一致性的前提下重绘场景。
  • 局部编辑 — 在左侧面板上传底图后，用画笔涂抹需要修改的区域，再描述修改内容，AI 只改涂抹区域。
  • 图片比例 — 支持 6 种比例：自动（auto）、1:1 方图、16:9 宽屏、9:16 竖屏、3:4 竖版、4:3 横版。默认自动，由 AI 根据参考图或场景决定。
  • 生成数量 — 下拉选择 1-4 张，默认 1 张。
  • 重绘幅度 — 0-1 之间，控制重绘强度。数值越大，AI 发挥空间越大；数值越小，越贴近原图。默认 0.5。

电商模式

专为电商主图设计，通过结构化参数（品类 + 卖点 + 字段）自动生成符合平台规范的高清主图。

  • 品类选择 — 支持多级级联，支持搜索。选到叶子节点后系统自动加载该品类的专属字段和卖点库。
  • 卖点 — 从预设卖点中单选，或点"➕ 添加卖点"自定义。每个卖点对应一套场景构图描述，直接影响生图结果。
  • 图片类型 — 主图（默认）/ SKU / 详情页。SKU 和详情页需要先上传参考主图。
  • 参考图 / 产品图 — 在配置弹窗底部上传，生图时优先于聊天区上传的图片。
  • 生成数量 — 下拉选择 1-4 张，默认 4 张。

开品模式

白底产品图 + 素材库创意参考的开品模式。先分析白底图，再从素材库选择创意图进行逐条融合生成。

  • 白底产品图 — 作为唯一产品主体，生成时必须保持造型、比例、颜色和结构一致。
  • 素材采集 — 看到有特色的图后上传，点击“生成提示词”，编辑后保存到开品数据库。
  • 引用数据库 — 从已保存素材中多选；每条素材都会和白底图、分析卡片提示词融合生成 1 张图。
  • 补充要求 — 可写目标人群、卖点、风格偏好，参与白底图分析和最终融合提示。
一致性 — 素材库图片只提供创意灵感，不作为新产品主体；主体始终以白底产品图为准。

视频模式

基于文字描述或参考图生成短视频（8 秒，720p）。适合制作产品展示视频、场景动态图。

  • 上传参考图后，AI 会以该图为起始帧生成动态视频。
  • 视频生成时间较长（约 2–5 分钟），请耐心等待进度条。
  • 生成完成后点 ⬇ 下载 mp4 文件。

电商模式详解

品类选择

点击"电商方案"按钮打开配置弹窗，品类选择器支持两种方式：

  • 级联浏览 — 逐级点击展开，点击条目本体即选中（任意层级均可停留），点击右侧 › 仅展开下一级不选中。
  • 搜索 — 在搜索框输入关键词，结果以"完整路径"形式平铺展示，点击即选中。支持中文输入法。

选中品类后，系统会自动加载该品类的专属字段（如安装方式、形态结构等）和卖点库。

卖点配置

卖点决定生图的核心场景构图。选中一个卖点后，下方"场景构图"区域会实时预览对应的构图描述。

  • 预设卖点来自品类数据库，选好品类后直接展示，无需先选形态结构。
  • 点"➕ 为当前形态添加卖点"可以自定义卖点，填写简称、文字要求、场景构图描述后保存，下次打开仍然可用。
  • 搜索框支持按卖点名称或 prompt 内容模糊匹配。

图片类型

类型用途是否需要参考图
主图电商列表主展示图，1:1 或 3:4可选（有参考图时主体一致性更强）
SKU白底/纯色背景产品展示图必须
详情页9:16 纵向详情页，每张参考图生成一张必须

专属字段说明

选中品类后会出现品类专属字段（标注"· 品类专属"），这些字段直接影响生图 prompt，建议按实际产品填写。部分字段仅在特定图片类型下生效（标注"· 仅 主图/SKU"等）。

开品模式详解

产品输入

开品模式需要先提供白底产品图，再从素材库选择创意参考：

输入项作用是否必填
白底产品图提供需要保持一致的产品主体必填
素材库图片提供造型、CMF、结构或场景创意参考生成时至少选 1 条
素材提示词保存素材时与图片一起入库，生成时作为创意说明保存素材时必填
纯文生图 — 如不上传图片，请选择 GPT-Image / 万相 / 通义 / 混元 等支持纯文生图的模型。Gemini 需要参考图才能生成。

素材数据库

素材数据库用于沉淀你看到的创意参考。每条数据包含一张图片和一段提示词：

操作说明
生成提示词AI 分析素材图的造型亮点、结构创新、CMF 和可借鉴创意点。
保存到数据库将素材图和提示词一起保存到本地 SQLite 数据库和图片目录。
引用数据库打开素材库界面多选，最终每条素材生成 1 张融合图。

融合规则

素材库融合时会遵循以下边界：

规则说明
主体一致白底产品图是唯一主体，素材图不能替换主体。
创意借鉴只提取素材图的造型语言、结构亮点、CMF、氛围或场景创意。
逐条输出选择 N 条数据库素材，就生成 N 张融合图。
禁止拼贴不能把白底图和素材图简单并列，必须融合成一个新的单体产品。

参考图使用

上传方式

有两个上传入口：

  • 聊天区拖拽/点击上传 — 适合临时参考，每次发送后清空。
  • 电商配置弹窗底部 — 参考图和产品图各一个框，保存配置后跨次发送保持不变，生图时优先于聊天区图片。

参考类别

参考图下方可以选择 AI 参考这张图的方式：

类别含义
构图参考以参考图的构图方式为主，产品/场景内容可自由发挥
分镜参考还原参考图的画面切分与视角逻辑
同类品参考以竞品图为参考，借鉴视觉亮点但保持本品牌风格
风格参考以参考图的色调/光影/氛围为主，产品本身保持原样
自定义自由填写参考方式描述，直接拼入生图指令
产品图 — 产品图框上传的是你的实际产品图，AI 会以此为主体进行场景重绘，保持产品造型/色彩/材质/logo 不变。

品类数据导入

系统内置了常用品类数据，如需添加自定义品类，可通过 xlsx 文件导入。

导入步骤

  1. 点击右上角头像菜单 → "导入品类 xlsx"
  2. 选择准备好的 .xlsx 文件（支持同时选多个）
  3. 等待导入完成，日志区域会显示处理结果
  4. 刷新页面后新品类即可在品类选择器中使用
注意 — xlsx 导入是增量合并，不会删除已有数据。同名品类会被新数据覆盖。

历史记录

点击右上角头像菜单 → "历史记录"，可以查看所有生图记录。

  • 每条记录显示生成时间、模式、使用的 prompt 和缩略图。
  • 点"重新生成"可以用相同参数再生成一次。
  • 已保存到画廊的图片会标注 💾 图标。
  • 删除历史记录不会删除已保存到画廊的图片。

常见问题

生图一直转圈没有结果？
检查网络连接和代理设置（设置 → 代理配置）。如果使用了代理，确认代理地址和端口正确。超时后系统会自动重试 2 次。
电商模式品类选择器是空的？
通常是打包版本的路径问题，升级到最新版本后可解决。如果仍然为空，尝试通过"导入品类 xlsx"手动导入品类数据。
生成的图片产品形状/颜色不对？
上传产品图并选择合适的参考类别（建议"同类品参考"或不选参考类别直接上传产品图），同时确保品类和卖点选择准确。
图片保存在哪里？
点 💾 后图片保存到"设置 → 文件保存位置"中指定的目录，默认是应用数据目录下的"生成结果"文件夹。
如何批量生成多张图？
电商模式下，详情页类型会按上传的参考图数量批量生成（每张参考图对应一张详情页）。其他类型目前每次生成 1 张，可以多次发送。
