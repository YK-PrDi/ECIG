# 自定义模式纯白底直出设计

## 目标

在自定义模式中，当用户同时选择“原图延展”、开启“跳过 AI 分析”、关闭“画面文案”，并在输入中明确要求白底图时，生成请求不得再拼接默认风格或类目营销提示词。

## 触发条件

以下条件必须同时成立：

1. 当前模式为自定义模式。
2. 改图风格为“原图延展”。
3. 已开启“跳过 AI 分析”。
4. 已关闭“画面文案”。
5. 用户输入包含明确白底意图，例如“白底图”“纯白背景”“白色背景”。

## Prompt 规则

触发后，最终 Prompt 仅由用户原始要求和固定的白底商品图约束组成：

- 上传图片是唯一产品主体。
- 产品外形、比例、颜色、材质、结构和识别细节保持一致。
- 使用纯白无缝背景和自然、真实的接触阴影。
- 单一产品主体，采用标准白底商品图构图。
- 不生成场景、道具、人物、装饰或营销氛围。
- 不生成标题、标签、文字、水印或额外 Logo。

该分支不拼接默认“原图延展”风格 Prompt、产品类目 Prompt 和类目负向 Prompt，避免卖点表达、场景化或营销化要求重新进入请求。

## 非目标

- 不修改 Gemini 分析和分析确认后的生成链路。
- 不改变其他自定义风格的 Prompt 拼接方式。
- 不改变未明确要求白底图时的直接生成逻辑。
- 不调整界面布局和交互控件。

## 测试

新增独立 Prompt 构造模块测试，覆盖：

- 四项条件成立时进入纯白底直出。
- 输出包含产品一致性、纯白背景、自然阴影、无文字、无道具约束。
- 输出不包含默认风格和类目营销 Prompt。
- 任一条件不成立时维持原有 Prompt 拼接。
